# React Web App Template

Client Side Web Application Template using React 19.

## React

- **Framework**: React 19 + TypeScript
- **Routing**: Wouter

## Data Management

- **State Management**: Jotai
- **GraphQL Client**: urql
- **Validation**: Zod

## Styling

- **Styling**: Tailwind CSS
- **Icon**: Lucide Icons

## Toolchain

- **Build tools**: Vite + SWC
- **Linter/Formatter**: Biome
